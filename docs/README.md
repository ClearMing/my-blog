## 简单的博客系统

我们要实现4个HTML页面，这四个页面构成一个完整的Web应用，即我们的个人博客。在开始项目之前，我们先创建项目目录并初始化几个文件。我们先给项目起一个名字，叫做myblog，在系统的命令行终端中创建这个目录，并初始化几个子目录和文件，用于后续开发。项目目录和文件结构如下：

```
myblog/
├── about.html  个人介绍
├── index.html  首页
├── links.html  友情链接
├── articles    文章目录
└── static  
    ├── css  外链的CSS 
    ├── image   图片
    └── js   外链的JS
```
